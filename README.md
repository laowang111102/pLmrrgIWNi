# 社区医疗管理系统 java622

## 项目概述

社区医疗管理系统是基于Spring Boot和Vue构建的健康管理平台，旨在为社区提供便捷、高效的医疗服务，同时适合初学者学习框架和技术实践。

## 技术栈

- 后端：Spring Boot
- 前端：Vue

## 功能模块

### 核心功能

- AI智能问诊：提供常见健康问题的解决方案。
- 预约挂号：方便患者预约医生，管理个人健康。
- 药品信息：详尽列出药品清单，指导合理用药。

### 系统角色

- **管理员**
  - 个人中心管理
  - 用户管理
  - 医生管理
  - 系统设置等

- **医生**
  - 个人中心
  - 出诊时段设置
  - 问诊记录查询
  - 患者管理

- **患者**
  - 医疗资讯浏览
  - 在线预约挂号
  - 门诊开药服务

## 系统优势

- **一体化服务**：整合医疗、预防、保健多项功能，打造现代化医疗服务机构。
- **在线交流**：支持用户之间分享健康经验和信息交流。

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：Node.js，现代浏览器

## 部署步骤

（以下为简化版步骤）

1. 准备后端运行环境：Java和MySQL。
2. 导入项目数据库结构。
3. 部署后端服务，启动Spring Boot应用。
4. 前端项目依赖安装：npm install。
5. 前端项目编译：npm run build。
6. 部署编译后的前端文件到Web服务器。
